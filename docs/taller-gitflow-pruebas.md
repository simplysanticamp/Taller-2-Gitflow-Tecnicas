# Documento de Pruebas

## 1. Descripcion del Sistema
El sistema permite validar el código institucional de un estudiante antes de registrarlo o procesarlo en la plataforma académica.  
El código debe cumplir reglas específicas de formato para garantizar consistencia y evitar errores en la base de datos.  
La validación verifica longitud, prefijo obligatorio y formato numérico.  
Si el código no cumple las condiciones, el sistema debe rechazarlo.
